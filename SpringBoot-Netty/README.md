# netty-tcp-demo

#### 介绍
基于netty的tcp长连接管理

#### 软件架构
基于netty、redis和springboot

#### 使用说明
1.  修改netty-tcp-client下的application.yml配置文件，修改redis的连接信息
2.  先后启动netty-tcp-server和netty-tcp-client项目，访问http://localhost:8088/demo/testOne开始测试
